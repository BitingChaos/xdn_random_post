== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the **xdn-random-post** folder to your **/wp-content/plugins/** directory or install it through the WordPress "Add New" function under the Plugins menu.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Add the **[random_post]** shortcode to any one of your pages, posts, or widgets on your site.

> **Example Usage**
>
> `[random_post]`
>
> `[random_post category="News"]`
>
